Introduction

Databases are a brilliant and easy way to keep track and store data, most suitably, large volumes of data.There are many types of databases, and they include:

  1. Relatonal Database
  2. NoSQL Database
  3. Cloud Database
  4. Columnar Database
  5. Wide column Database
  6. Object-oriented Database
  7. Key-value Database
  8. Hierarchical Database
  9. Document Database
  10. Graph Database
  11. Time series Database
Storing and organizing data in a databases, follows many rules, which often depend on the type of Database Management System (DBMS) being used, and the type of data being stored. This [mock] documentation will cover the MySQL Database using SQL
